Postman

Inserción de registros con POSTMAN con autorización tipo Basic

Haremos una petición de tipo POST a la URL http://localhost:3333/api/inmueble la cual está corriendo en nuestra aplicación.

Debemos agregar datos de autorización en la pestaña Authorization con los datos del usuario registrado de nuestra app

Luego en la pestaña Body enviaremos el archivo JSON con los datos que espera recibir InmuebleController y presionaremos SEND

Consulta GET a la Base de datos

Usaremos la misma URL http://localhost:3333/api/inmueble sin embargo cambiaremos el tipo de petición de POST a GET y manteniendo el tipo de autorización basic Auth con el usuario utilizado anteriormente.

Eliminación del registro método DELETE

Para eliminar el registro necesitamos usar el método DELETE, además del id del registro a eliminar, en este caso el id 1, usando la URL http://localhost:3333/api/inmueble/1 que tenemos asignada en nuestro InmuebleController.
